# wplex-garagem-spring-boot

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

WPLEX Garage Project with Spring Boot Application 1.5.<br>
<b>Demo Application running</b> on [https://labix.orionxf.com.br/](https://labix.orionxf.com.br/)

- Support Spring MVC and RESTful webservices.<br>
- The application is deployed on [AWS](https://aws.amazon.com/pt/) Plataform.<br>
- Spring Datasource uses HikariCP for Database Connection Pooling.<br>
- Relational Database is PostgreSQL with Spring Data and Hibernate.<br>
- REST API Documantation is auto-generated by Swagger Maven Plugin.<br>
- WEB-MVC Controller with Thymeleaf Template Engine.

Swagger 2.0 Documentation on https://labix.orionxf.com.br/api-portal/swagger/1.0/index.html

### Requirements

- Java 1.8 or later.
- PostgreSQL Server 9.6 or later.
- AWS account for EC2, Route53.

### Configuration and Package

To compile the binary package run: mvn clean package -DskipTests
On the IDE, field VM arguments input: -Dspring.profiles.active=dev
