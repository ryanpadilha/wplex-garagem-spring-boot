# wplex-garagem-spring-boot

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

WPLEX Garage Project with Spring Boot Application 1.5.<br>
<b>Demo Application running</b> on [https://wplex-app.herokuapp.com/](https://wplex-app.herokuapp.com/)

- Support Spring MVC and RESTful webservices.<br>
- The application is deployed on [Heroku](https://www.heroku.com/) Plataform.<br>
- Spring Datasource uses HikariCP for Database Connection Pooling.<br>
- Relational Database is MySQL with Spring Data and Hibernate.<br>
- REST API Documantation is auto-generated by Swagger Maven Plugin.<br>
- WEB-MVC Controller with Thymeleaf Template Engine.

Swagger 2.0 Documentation on https://wplex-app.herokuapp.com/api-portal/swagger/1.0/index.html

### Requirements

- Java 1.8 or later.
- MySQL Server 5.1 or later.
- Heroku free account with verified user (for MySQL).
